# EcoTrend

## Sobre o projeto
O **EcoTrend** é um e-commerce fictício desenvolvido com foco em **produtos sustentáveis e ecológicos**, incentivando um estilo de vida mais consciente e responsável.  
A proposta do projeto é criar uma interface moderna, organizada e responsiva para apresentar produtos de diferentes categorias, unindo estética, usabilidade e preocupação ambiental. :contentReference[oaicite:1]{index=1}

Este projeto foi desenvolvido para o **Check-Point 04** da disciplina de **Front-end Design** do curso de **Engenharia de Software**. :contentReference[oaicite:2]{index=2}

---

## Objetivo
Desenvolver a interface responsiva de um e-commerce especializado em produtos sustentáveis, utilizando:

- **HTML**
- **CSS**
- **Bootstrap**
- **Grid Layout**

O projeto busca aplicar conceitos de estruturação de páginas, responsividade, organização visual e personalização de componentes do Bootstrap. :contentReference[oaicite:3]{index=3}

---

## Proposta da aplicação
A EcoTrend foi idealizada como uma loja virtual voltada para consumidores que valorizam produtos com proposta mais sustentável.  
A interface foi construída para oferecer uma navegação simples, visual agradável e organização clara das informações, destacando produtos, categorias e formas de contato.

---

## Funcionalidades
O projeto conta com as seguintes páginas e seções:

### Página Inicial (Home)
- Barra de navegação com logo, menu e carrinho
- Carrossel de banners em destaque
- Seção de produtos em destaque
- Área com informações sobre a marca
- Rodapé com contatos e redes sociais

### Página de Categorias
- Exibição de produtos organizados em grid
- Estrutura preparada para navegação por tipos de produto

### Página de Produtos
- Listagem de produtos
- Cards com imagem, nome, descrição e botão de compra

### Página de Contato
- Formulário estilizado para contato com o cliente

### Página de Carrinho
- Estrutura visual para exibição dos produtos selecionados
- Resumo de compra e área de checkout

Esses elementos seguem a proposta do enunciado, que solicita home, categorias, detalhes do produto, contato e footer informativo. :contentReference[oaicite:4]{index=4}

---

## Categorias de produtos
A EcoTrend trabalha com a proposta de produtos ligados à sustentabilidade, como:

- Roupas e acessórios sustentáveis
- Produtos de beleza e cuidados pessoais naturais
- Itens para casa sustentáveis
- Produtos com proposta de tecnologia verde :contentReference[oaicite:5]{index=5}

---

## Tecnologias utilizadas
- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **Bootstrap Icons**
- **Google Fonts**

O Bootstrap foi utilizado para facilitar a construção de componentes como **cards, carrossel e formulários**, enquanto o CSS foi usado para personalizar a identidade visual do projeto. :contentReference[oaicite:6]{index=6}

---

## Responsividade
Um dos principais objetivos do projeto foi garantir que a interface funcionasse bem em diferentes tamanhos de tela.  
Para isso, foram aplicados conceitos de:

- **layout responsivo**
- **media queries**
- **grid**
- **flexbox**
- **componentes adaptáveis do Bootstrap** :contentReference[oaicite:7]{index=7}

Assim, a navegação e a visualização dos elementos foram ajustadas para desktop, tablet e smartphone.

---

## Estilização
A identidade visual da EcoTrend foi construída com foco em transmitir:

- natureza
- leveza
- bem-estar
- consumo consciente

Foram utilizadas cores em tons de verde e off-white para reforçar a proposta ecológica da marca, além de estilos personalizados aplicados sobre os componentes do Bootstrap, conforme solicitado no projeto. :contentReference[oaicite:8]{index=8}
